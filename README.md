# 🚔 Crime Data Analysis Dashboard 📊

## 📌 Project Overview

This project presents a structured **crime analysis solution using Power BI**, designed to deliver clear and actionable insights through multiple focused dashboards.

The dataset consists of **100,000+ crime records**, transformed into an interactive and user-friendly reporting system.

---

## 🎯 Project Approach

✔ Divided into multiple dashboards for clarity
✔ Focus on simplicity and readability
✔ Designed for effective data storytelling

---

## 🏠 Dashboard 1: Crime Overview

### 📊 Visuals

* Crime Trend (Line Chart)
* Crime by State (Map)
* Crime Type Distribution (Treemap)
* Area Type Split (Donut Chart)

🎯 **Purpose:**
Provides a quick summary of overall crime patterns
